Git Practice

As a Software Developer it is good to learn about version control system. The most popular of distributed version control system is Git.

Step to get started

  1. Create a folder and give it name.
  2. Initialise your Git repository with the command - git init
  3. Open the folder using git bash
  4. Create files example index.html and index.css using the command touch index.html and touch index.css, in each case press enter to creat the file.
  5. Open the folder to verify if the files are physically in there.
  6. You are again verify using the git commmand, git status -
    the files should be in red color.
  7. These files are untracked and has to be added to git by using this command git add index.html enter ; git add index.css -- This command will add one file each . But to add more files at once --use git add . or git add * commands
  8. Open the files created and open in your favourite text editor and add some content and styles and save.
  9. Then commit the new files.
